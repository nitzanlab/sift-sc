Installation
============

SiFT installation
-----------------
Install `SiFT` in one of the following ways:

Through **pip**::

    pip install sift-sc

Nightly version - clone this repo and run::

    git clone https://github.com/nitzanlab/sift-sc
    pip install .

For development - clone this repo and run::

    git clone https://github.com/nitzanlab/sift-sc
    pip install -e .[dev,docs]
